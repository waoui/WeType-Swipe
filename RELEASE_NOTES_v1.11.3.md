# 微信输入法下滑快捷键 v1.11.3

本版本修复微信输入法 3.5.2 部分构建中的原生面板兼容问题，并修正数字／符号页面错误显示下滑标签的问题。

## 修复内容

- 修复下滑打开微信输入法原生剪贴板失效。
- 修复下滑打开快捷发送／常用语失效。
- 原生工具栏入口不再依赖单一混淆监听器类名，改为“已知字段快速匹配 + 结构特征回退”。
- 修复数字／符号页面错误显示“剪贴、快捷、选前、选后、段首”等 QWERTY 功能标签。
- 手写找字及其他手写键盘界面不再显示标签或执行下滑动作，避免干扰手写操作。
- 切换字母、数字和符号页面时，根据当前可见按键重新判定并缓存标签。

## 兼容性

- Android 8.0 及以上
- LSPosed
- 微信输入法 3.5.0、3.5.2
- 本次真机验证环境：Android 16、微信输入法 3.5.2

## 升级说明

可直接覆盖安装 v1.11.2。升级后建议强制停止并重新启动微信输入法；原有键位映射、自定义标签和开关配置保持不变。

## 发布验证

- 单元测试、Android Lint、Release 构建通过
- 永久发布证书 SHA-256 校验通过
- 26 键与九宫格原生剪贴板、快捷发送／常用语真机验证通过
- 两级数字／符号页面标签显示验证通过
- 正常点击、长按、滑行输入未发现回归
