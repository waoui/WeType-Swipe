# 更新日志

## v1.7.0

- 修复 v1.6 下滑手势失效问题。
- 恢复已实机验证的 `View.dispatchTouchEvent` 触摸链路。
- 仅在微信输入法进程内安装 Hook。
- 首次识别键盘类后缓存类型，其他 View 只做快速类型判断。
- 未绑定按键完全放行，不进入后续手势计算。
- 支持指定多个按键禁用下滑。
- 修复关闭震动后配置未及时刷新的问题。
- 使用 Modern Xposed API 102。
- 静态作用域仅声明 `com.tencent.wetype`。
- 从本版开始固定发布签名，后续版本可覆盖安装。

## v1.6.0

- 删除段首、段尾动作。
- 增加禁用下滑按键配置。
- 尝试迁移到精确键盘 Hook；该版本实际出现下滑失效，已在 v1.7.0 回退修复。

## v1.5.0

- 删除全复制、全剪切。
- 增加隐藏桌面图标功能。
- 增加 LSPosed 静态作用域声明。

## v1.4.0

- 增加动态按键映射。
- 优化配置缓存和触摸事件处理。

## v1.3.0

- 首个实机验证可用版本。
- 通过系统触摸分发入口识别微信输入法键盘。
