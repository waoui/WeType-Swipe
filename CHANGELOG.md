# 更新记录

## v1.11.4

- 新增“撤销”和“重做”下滑动作，支持 26 键与九宫格绑定
- 动作编号追加为 14、15，不改变既有动作编号和用户配置
- 使用 Android 标准编辑器撤销／重做指令，不保存用户输入内容
- 触发前结束当前组词状态，由目标 App 决定撤销粒度和历史顺序
- 保持数字、符号、手写键盘及密码输入框隔离逻辑不变
- 保持 v1.11.3 原生面板、标签和微信输入法 3.5.2 兼容修复不变

## v1.11.3

- 修复部分微信输入法 3.5.2 构建中原生剪贴板、快捷发送／常用语入口失效
- 原生工具栏回调由固定混淆类名匹配改为已知字段快速匹配和结构特征回退
- 修复数字／符号页面错误显示 QWERTY 下滑功能标签
- 手写找字及其他手写键盘界面不再显示或触发下滑快捷动作
- 标签缓存加入当前可见按键身份，切换键盘层后不会复用旧标签
- 保持原有配置格式、默认键位和 3.5.0 兼容逻辑不变

## v1.11.2

- 修复微信输入法 3.5.2 下滑打开原生剪贴板失效
- 修复微信输入法 3.5.2 下滑打开快捷发送／常用语失效
- 同时兼容微信输入法 3.5.0 与 3.5.2 的原生工具栏入口
- 增加原生工具栏载体匹配日志，便于后续版本兼容排查

## v1.11.1

- 新增“复制全部”和“剪切全部”，并在复制后恢复原光标或选区
- 26 键和九宫格支持长按按键设置显示标签
- 标签支持自动、自定义和隐藏三种模式，自定义内容最多 4 个字符
- 保存配置后立即刷新原生单键文字缓存

## v1.11.0

- 在 26 键和九宫格键帽底部显示已绑定的下滑功能
- 新增按键功能文字和下滑触发提示独立开关
- 接入微信输入法原生单键绘制链，适配横竖屏、悬浮与分离键盘
- 缓存按键模型和显示文字，移除周期扫描与布局轮询

## v1.10.0

- 新增原生剪贴板与快捷发送／常用语动作
- 缓存原生工具栏命令载体与反射成员
- 优化横竖屏、悬浮键盘和工具栏切换时的缓存失效处理

## v1.9.4

- 新增“段首”和“段尾”
- 新增“选至段首”和“选至段尾”
- 通过光标选区与周边文本实现段落边界计算
- 修复 Android 15/16 顶栏与状态栏重叠
- 26 键设置改为标准 QWERTY 键盘布局
- 九宫格设置改为 `1–9` 键盘布局，不显示 `0`、`*`、`#`
- 删除测试阶段的段落诊断日志
- 保持 v1.8.6 九宫格配置同步和手势逻辑不变

## v1.8.6

- 修复九宫格配置在微信输入法子进程中无法稳定生效的问题
- 保存配置时发送完整配置快照
- 微信输入法各进程从目标应用私有缓存读取同一份配置

## v1.8.0

- 新增九宫格 `2–9` 独立下滑映射
- 新增九宫格独立触发距离

## v1.7.0

- 恢复稳定的 `View.dispatchTouchEvent` 触摸链路
- 使用 Modern Xposed API 102
