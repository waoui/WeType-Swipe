# 微信输入法下滑快捷键 v1.11.4

本版本新增可绑定的“撤销”和“重做”下滑动作。

## 新增功能

- 26 键和九宫格均可绑定“撤销”和“重做”。
- 使用 Android 标准编辑器菜单指令执行撤销与重做。
- 触发前结束当前输入法组词状态，再交由目标 App 的输入框处理。
- 不保存输入内容，不建立模块私有撤销历史。

## 行为说明

撤销粒度由目标 App 的编辑器决定。在拼音直接显示于输入框的模式下，中文上屏可能被记录为“拼音输入”和“拼音替换为中文”两个步骤，因此可能出现：

`中文 → 拼音 → 空白 → 拼音 → 中文`

这属于正常撤销／重做顺序。不同 App、网页富文本、小程序和自绘编辑器的支持情况可能不同。

## 兼容性

- Android 8.0 及以上
- LSPosed
- 微信输入法 3.5.0、3.5.2
- 本次真机验收环境：Android 16、微信输入法 3.5.2
