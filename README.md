# 微信输入法下滑快捷键（WeType Swipe）

为安卓版微信输入法增加可自定义的字母键下滑操作。当前支持 **全选、剪切、复制、粘贴**，基于 LSPosed / Modern Xposed API 102。

> 当前版本：v1.7.0 公测版

## 功能

- 任意 A-Z 字母可绑定下滑动作
- 支持全选、剪切、复制、粘贴
- 正常点击按键仍输入原字母
- 未绑定按键完全放行
- 可指定多个按键禁用下滑
- 可调节下滑触发距离
- 可关闭模块主动触发的震动
- 支持隐藏桌面图标
- 静态作用域仅为微信输入法 `com.tencent.wetype`
- 配置常驻内存，触摸过程中不读取配置文件
- 不修改微信输入法 APK

## 默认映射

| 按键 | 下滑动作 |
|---|---|
| Z | 全选 |
| X | 剪切 |
| C | 复制 |
| V | 粘贴 |

## 使用方法

1. 安装 APK。
2. 在 LSPosed 中启用本模块。
3. 确认作用域为“微信输入法”。
4. 重启微信输入法进程或重启手机。
5. 打开模块并设置按键与动作。

- 微信输入法：3.5.0
- Android：16
- 架构：arm64
- 框架：LSPosed

其他系统版本、键盘布局和微信输入法版本尚未全部验证。

## 注意事项

- 当前主要适配微信输入法 26 键拼音和英文键盘。
- 九宫格、手写、符号面板未专项适配。
- 关闭模块震动后，如果仍有按键震动，可能是微信输入法自身的按键反馈。
- 模块不联网，不收集、上传或保存输入内容。
- 未绑定按键不进入手势处理流程。
